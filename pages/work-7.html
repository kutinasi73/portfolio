  • 制作時間

    20.3時間

    (企画:2時間 / デザイン：6.3時間 / コーディング：12時間)

  • 使用技術

    ･Design: Figma, Photoshop

    ･Frontend: HTML5, CSS3

    ･Others: PHP(WordPress静的化), GitHub

  • 制作担当

    ディレクション / コピーライティング / デザイン / コーディング(全工程)

  • 対応機種

    レスポンシブ（PC / スマートフォン）

  • ターゲット

    20〜30代のアパレル･クリエイティブ職･フリーランスなど､自己表現を重視する感度の高い女性｡

  • 課題と解決

    課題：

    インスタの画像だけでは伝えきれない､サロン全体の｢センス｣や｢空気感｣への信頼構築｡

    解決：

    1. 非対称グリッドレイアウト :雑誌のコラージュのような動きのある配置で､型にハマらない｢ニュアンスネイル｣の柔軟さを体現｡

    2. 直感への訴求:説明文を最小限にし､ビジュアルと言葉の断片で｢この人にお任せしたい｣と思わせる情緒的アプローチ｡

    3. SNSとの一貫性:SNSの熱量をそのまま予約に繋げるため､SNSのデザインコード（配色･質感）をサイト内にも反映｡

  • 制作背景

    自主制作｡Gardenia-Aと同一店舗を題材に､ターゲットを若年層･トレンド層にシフト｡感性に訴えかけるデザインがコンバージョンに与える影響を検証｡

  • デザインのこだわり･工夫

    リップカラーをイメージした配色と､規則的なグリッドと､あえてグリッドを崩したファッショナブルなレイアウト｡

    Font：Inter/ Shippori Mincho / Bodoni Moda / Playfair Display（和モダン×洗練）

    Color：#fff1f1 / #bf5a5a / #840000（大人の色気とトレンド感）

  • 実装のこだわり･工夫

    1. CSS Gridを駆使し､複雑なレイアウトを保ちつつ､レスポンシブ時の可読性も維持｡

    2. 背景の質感や文字のコントラストを微調整し､ネイル写真が最も際立つトーンを追求｡

  • 振り返り

    良かった点：

    Aパターン（信頼）に対し､Bパターン（感性）という明確なコンセプトの打ち分けができ､制作の引き出しを広げられた｡
