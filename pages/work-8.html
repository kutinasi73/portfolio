    20.6時間

    (企画:2時間 / デザイン：4.6時間 / コーディング：14時間)

  • 使用技術

    ･Design: Figma, Photoshop

    ･Frontend: HTML5, CSS3, JavaScript (Vanilla JS)

    ･Others: PHP(WordPress静的化), GitHub

  • 制作担当

    ディレクション / コピーライティング / デザイン / コーディング(全工程)

  • 対応機種

    レスポンシブ（PC / スマートフォン）

  • ターゲット

    30〜60代の建設業経営者･雇用主｡深刻な人手不足に悩み､解決策を論理的に比較検討している層｡

  • 課題と解決

    課題：

    申請手続きの不透明さと､行政書士という存在への｢敷居の高さ｣による離脱｡

    解決：

    1. ロジックの可視化 :採用までのステップを論理的な図解で提示｡｢何を･いつまでに･いくらで｣を明確化｡

    2. 離脱防止設計:ヒートマップ分析を想定し､疑問が生まれるタイミング（価格･フロー）で適切なコンテンツを配置｡

    3. 権威性の構築:実績と専門性を前面に出し､｢この事務所に任せれば安心｣という実利的な信頼を醸成｡

  • 制作背景

    堅実･誠実を体現するネイビー基調｡情報の階層を明確にし､多忙な経営者の｢速読｣をサポートする機能的レイアウト｡

  • デザインのこだわり･工夫

    余白を活かした｢静寂なサロン空間｣を演出するレイアウト｡装飾を削ぎ落とし､清潔感とプロフェッショナルな品格を表現｡

    Color：#F2F4F6 / #C5A059 / #1F3A5F（堅実･清潔･スピード感）

  • 実装のこだわり･工夫

    1. ライブラリに頼らないVanilla JSでの実装により､現場や移動中の通信環境でもストレスのない高速表示を追求｡

    2. フォームまでのスムーズなスクロール導線と､視認性の高いCTA配置｡
