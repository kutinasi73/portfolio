希少性を価値に変える予約専用LP

SNSでの話題性を確実に実売（予約）へ繋げるためのクロージング特化型LPです｡

  • 制作時間

    23.6時間

    (企画:2時間 / デザイン：7.8時間 / コーディング：13.8時間)

  • 使用技術

    ･Design: Figma, Photoshop

    ･Frontend: HTML5, CSS3

    ･Others: PHP(WordPress静的化), GitHub

  • 制作担当

    ディレクション / コピーライティング / デザイン / コーディング(全工程)

  • 対応機種

    レスポンシブ（PC / スマートフォン）

  • ターゲット

    20〜40代の｢丁寧な暮らし｣志向層｡SNSでパン屋をフォローしているが､来店時の売り切れを懸念して購買に踏み切れていない潜在顧客｡

  • 課題と解決

    課題：

    SNSで｢いいね｣は増えるが予約に繋がらない｡原因は｢売り切れるかも｣という不安と､｢写真だけでは価値が伝わらない｣という情報不足｡

    解決：

    1. 希少性の数値化 :週末限定30本｣を全セクションで一貫して使用し｢手に入りにくい = 価値が高い｣という認知を構築｡

    2. 五感への訴求:｢パチパチという音｣｢芳醇な香り｣など音や香りを想起させるコピーで写真以上の体験を補完｡

    3. 納得感の醸成 :酵母･熟成･成形の3ステップの熟成工程を可視化し､高単価への納得感を構築｡

  • 制作背景

    自主制作｡SNSで集客しているが予約転換率が低い店舗の課題を解決するLPを自主制作｡クライアントワークを想定したリアルな課題設定で制作｡

  • デザインのこだわり･工夫

    余白を贅沢に使ったエディトリアルレイアウト｡数字（30 / 48 / 3）を大きくアクセントに使い､スキャンするだけで希少性が伝わるよう設計｡パンの重厚感とブランドの品格を表現｡

    Font：Noto Serif JP / DM Serif Display（信頼･高級感･手仕事感を両立）

    Color：#FAF2E8 / #D9C0A3 / #593D2D (焦げ茶（焼き色）､アイボリー（小麦粉）)

  • 実装のこだわり･工夫

    1. CTAボタンを固定表示し､スクロール中どこでも予約できる設計

    2. 無限ループスライダーの配置（ライブラリ不使用）

  • 振り返り

    良かった点：

    コピーと構成を先に固めてからデザインに入ったことで､ビジュアルとメッセージの一貫性が保てた｡

    改善したい点：

    予約フォームが未実装のため､実際のCVR計測ができない｡次回はGA4 + Googleフォームとの連携まで含めて設計したい｡
