朝のひとひとときを､最高の体験に｡
ライ麦パン専門カフェの集客LP

｢ライ麦パン＝食べにくい｣という先入観を払拭し､モーニング需要を喚起する集客LP｡UI実装をライブラリ不使用のフルスクラッチで制作したLPです｡

  • 制作時間

    29時間

    (企画:0.5時間 / デザイン：8.5時間 / コーディング：20時間)

  • 使用技術

    ･Design: Figma, Photoshop

    ･Frontend: HTML5, CSS3, JavaScript (Vanilla JS)

    ･Others: GitHub

  • 制作担当

    ディレクション / コピーライティング / デザイン / コーディング(全工程)

  • 対応機種

    レスポンシブ（PC / スマートフォン）

  • ターゲット

    20〜30代の健康意識が高い女性｡忙しい日々の中で｢少し贅沢で心地よい朝の時間｣を求めている層｡

  • 課題と解決

    課題：

    ライ麦パンに対する｢硬い･酸っぱい｣というネガティブなイメージによる集客機会の損失｡

    解決：

    1. 体験の視覚化 :単なるメニュー紹介に留まらず､光が差し込む店内の雰囲気やシズル感のある写真を多用し､理想のモーニング体験を想起させる構成｡

    2. UI/UXの最適化:多忙な朝に情報を探すユーザーを想定し､営業時間やアクセス方法を即座に把握できるシンプルな情報設計｡

    3. 情緒的なアプローチ :余白を贅沢に使い､北欧モダンな世界観で｢清潔感｣と｢温かみ｣を両立｡

  • 制作背景

    自主制作｡健康志向の20〜30代女性をターゲットに､設計･デザイン･実装を一貫して行い､フロントエンドの実装力向上を目指したプロジェクト｡

  • デザインのこだわり･工夫

    ホワイトスペースを意識したミニマルなレイアウト｡スキャン（流し読み）でも｢美味しさ｣と｢居心地の良さ｣が伝わるよう､セクションごとにミリ単位で余白を調整｡

    Font：Inter / Indie Flower / M PLUS 1（モダンで読みやすく､都会的な洗練さを演出）

    Color：ゴールデンブラウン（ライ麦） / ホワイト / チャコール（落ち着き）

  • 実装のこだわり･工夫

    1. フルスクラッチ実装 ：外部ライブラリに頼らず､ハンバーガーメニュー､スライダーを自作し､軽量化と高い制御性を実現｡

    2. SVG :装飾にSVGを採用し､ページを開いた瞬間にブランドの個性が動的に伝わる演出を付与｡

    3. 背景デザインの数ピクセルへの執着 :セクションタイトルの背景にある｢真円｣の配置に最も時間を費やした｡異なるデバイス（PC/スマホ）や解像度でも､文字との黄金比を崩さず､かつ｢温かみ｣を感じさせる絶妙な位置（transform と pseudo-elements による微調整）を追求し､サイト全体の完成度を底上げした｡

  • 振り返り

    良かった点：

    構造（HTML）･装飾（CSS）挙動（JS）を明確に分離して管理することで､保守性の高いコードを書く意識が定着した｡

    改善したい点：

    アニメーションのパフォーマンスなど､より高度なビジュアル表現と実行速度の両立を次回の課題としたい｡
